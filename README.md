## 🚀 Overview

The Intelligent Medical Assistant is a full-stack system designed for clinics and medical specialists. It streamlines daily operations such as patient registration, appointment scheduling, and medical documentation, while providing AI-powered tools to support diagnosis and decision-making.

The platform connects assistants, doctors, and administrators through dedicated dashboards, improving efficiency, accuracy, and overall patient care.

## ✨ Features
### 🤖 AI Capabilities
Feature	Description
Voice-to-Report	Converts doctor speech into structured medical reports & prescriptions
Medical NLP	Extracts symptoms, diagnoses, and generates summaries (SOAP format)
X-ray Analysis	AI-assisted detection from radiology images
Skin Disease Detection	Computer vision for visible conditions
Lab Report Interpretation	Converts lab results into clear medical insights
### 🏥 System Features
Patient registration & appointment scheduling
Electronic Health Records (EHR) management
Full patient history tracking (chronic diseases, visits, medications)
Bilingual reports (Arabic / English)
Role-based dashboards (Doctor / Assistant / Admin)
Secure data handling with access control
### 🖥️ Interfaces
Assistant Dashboard → Manage patients & appointments
Doctor Dashboard → View history, record voice notes, get AI support
Admin Panel → Manage users, monitor system, ensure security
